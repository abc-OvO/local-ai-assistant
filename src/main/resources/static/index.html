平台功能

面向 RAG 项目演示的完整功能

RAG

多文档 RAG

从全部已索引文档中检索相关片段，并在回答中保留来源依据。

SEM

语义搜索

结合本地向量检索和关键词信号，减少仅靠字面匹配造成的漏检。

MEM

会话记忆

通过 sessionId 管理多轮上下文，让后续提问能够延续上一轮主题。

EMB

本地 Embedding

Ollama 在本地生成 Embedding，让知识库索引可以直接在本机运行。

API

Provider 切换

通过 Router 在 Kimi 和 Ollama 之间切换生成服务，并支持后续扩展。

OBS

可解释检索

前端展示 retrievedChunks 和评分明细，便于调试 RAG 检索质量。
