# 项目截图占位说明

该目录保存 README 中展示的项目截图和架构图。

| 文件名 | 内容 |
| --- | --- |
| `landing.png` | 本地运行的 Landing Page 首屏 |
| `workspace.png` | 本地运行的 Chat-first Workspace |
| `rag-demo.png` | 真实 Auto RAG 问答、Sources 和检索评分 |
| `architecture.png` | 与 README 系统架构一致的导出图 |

维护截图时请注意：

- 使用真实运行数据，但避免包含 API Key、数据库密码、个人路径或敏感文档内容。
- 保持相同浏览器尺寸和浅色 / 深色主题，推荐宽度不低于 1440 px。
- 提交前检查图片体积，单张建议控制在 1 MB 左右。
