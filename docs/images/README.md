# 项目截图占位说明

该目录用于保存 README 中展示的真实项目截图。目前未提交截图文件，避免使用生成图片代替实际运行效果。

请在本地启动项目后，按以下名称补充 PNG 文件：

| 文件名 | 建议截图内容 |
| --- | --- |
| `landing.png` | 首页完整首屏，展示项目定位和进入 Workspace 的入口 |
| `workspace.png` | Chat-first Workspace 全局界面，包含聊天区、文档区和模型状态 |
| `rag-demo.png` | 一次真实 RAG 问答，展示回答、Sources 和检索评分 |
| `architecture.png` | 将 README 中 Mermaid 系统架构图导出为清晰 PNG |

截图建议：

- 使用真实运行数据，但避免包含 API Key、数据库密码、个人路径或敏感文档内容。
- 保持相同浏览器尺寸和浅色 / 深色主题，推荐宽度不低于 1440 px。
- 截图完成后，取消 README 中对应图片引用的 HTML 注释。
- 提交前检查图片体积，单张建议控制在 1 MB 左右。
